# SOC Phishing Email Analysis Lab

This project demonstrates how a SOC analyst investigates a phishing email using Kali Linux tools.

## Tools Used

Kali Linux
Python
WHOIS
DNS analysis

## Investigation Steps

1. Email inspection
2. URL extraction with Python
3. Domain investigation using WHOIS
4. DNS resolution analysis
5. IOC identification
6. Incident report generation

## Skills Demonstrated

Phishing detection
Threat intelligence
SOC investigation workflow
Incident documentation
